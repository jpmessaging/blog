新しい Outlook for Windows のメール テンプレート機能の紹介

Last Update: 2026-01-29

こんにちは。日本マイクロソフト Exchange & Outlook サポート チームの杉山です。
本記事では、新しい Outlook for Windows (以下、新しい Outlook) に 2025 年 11 月に新機能として追加されたメール テンプレート機能をご紹介します。
メール テンプレートを活用することにより、共通の情報が含まれるメッセージを簡単に作成できますので是非活用ください！

メール テンプレートの作成方法

新しい Outlook では以下の手順でメール テンプレートを作成できます。

  1. 新しいメール メッセージを作成し、共通情報が含まれるメールを作成します。
  2. メールを作成後、[ファイル] - [テンプレート] より [メールをテンプレートとして保存] をクリックします。
    ※ [メッセージ] -タブより [メール テンプレート] - [メールをテンプレートとして保存] からも実行できます。
  3. [メールをテンプレートとして保存] ダイアログより、メール テンプレートに名前を付け、[保存] をクリックします。

メール テンプレートからメールを作成する方法

保存したメール テンプレートは新規メール作成時に使用できます。

  1. [新規] - [テンプレートからのメール] をクリックします。
  2. 使用するテンプレートを選択します。


テンプレートの設定

保存されたテンプレートは [設定] - [メール] - [テンプレート] より確認できます。
※ 新しい Outlook で保存したテンプレートは Outlook on the web でも使用できます。

右上の [テンプレートを追加] より OFT ファイルをインポートすることもできます。
これにより、従来の Outlook for Windows (以下、従来の Outlook) で利用していたテンプレートの内容をもとに新しい Outlook のテンプレートを追加できます。

Tips
画面内に記載の通り従来の Outlook で使用していたテンプレートは %APPDATA%\Microsoft\Templates に保存されていますので、エクスプローラーなどで OFT ファイルを確認できます。

メール メッセージのテンプレートの作成についてはこちらの公開情報もご一読ください。

本情報の内容（添付文書、リンク先などを含む）は、作成日時点でのものであり、予告なく変更される場合があります。
