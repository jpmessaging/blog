Microsoft 365 Copilot Chat のピン留めの Outlook での表示イメージ

Last Update: 2025-12-23

こんにちは。日本マイクロソフト Exchange ＆ Outlook サポート チームの山本です。

本記事では、以前よりお問い合わせをいただくことがある Outlook における Microsoft 365 Copilot Chat (以下、Copilot Chat) のピン留め機能について説明します。

Microsoft 365 Copilot Chat のピン留めについて

現在 Copilot Chat は、Microsoft 365 Copilot ライセンス (以下、Copilot ライセンス) の有無 (※) に関わらず、Web、デスクトップ、モバイルといった各プラットフォーム上の Microsoft 365 の Outlook でピン留めして利用することが可能になっています。

Microsoft 365 アプリでMicrosoft 365 Copilot Chatをピン留めする

以前は、Copilot ライセンスが割り当てられたユーザーのみ、Copilot Chat が自動的にナビゲーションバーにピン留めされて利用が可能でした。
現在では、Microsoft 365 Copilot ライセンスの有無に関わらず、様々なライセンスを持つ Entra アカウント ユーザーでも Copilot Chat の機能が利用できるようになっています。これに伴い、Copilot ライセンスを未割り当てのユーザーを対象にした Copilot Chat のピン留めを制御する管理者向けの機能が展開されています。
この制御に関する設定項目は、Microsoft 365 管理センターの [Copilot] - [設定] - [ユーザー アクセス] - [Microsoft 365 Copilot Chat をピン留めする] にあり、既定値では [Microsoft 365 アプリに Copilot Chat をピン留めします (推奨)] が設定されています。
既定値のまま利用されている環境では、Copilot ライセンスが未割り当てのユーザーにおいても各 Microsoft 365 アプリで Copilot Chat が自動で表示され、ピン留めされます。

(※) Copilot ライセンスが未割り当てであっても Copilot Chat をご利用いただけるアカウント
2025 年 10 月 30 日時点で、以下いずれかのライセンスを持つ Entra アカウント ユーザーでは、Copilot Chat が利用可能です。

・Microsoft 365 A1/A3/A5 (学生向け MA3/MA5、教員向け MA3/MA5、MA3/MA5 学生利用特典を含む)
・Microsoft 365 Business Basic/Business Standard/Business Premium
・Microsoft 365 E3/E5
・Microsoft 365 F1/F3
・Microsoft Teams/Teams Enterprise/Teams Essentials/Teams Rooms
・Office 365 A1/A1 Plus/A3/A5
・Office 365 E1/E1 Plus/E3/E5
・Office 365 F3

各 Outlook における Copilot Chat のピン留めの画面イメージ

Outlook は様々なプラットフォームで利用可能ですが、Copilot Chat のピン留めが有効な場合の各画面イメージを紹介します。
なお、以下の各画面イメージは 2025 年 12 月 1 日時点の情報です。
Copilot 機能は日々更新されており、実際にご利用の際は画面表示が本記事と異なる場合がございます。
